---
description: Building frontend assets using Laravel Mix
---

1. Ensure dependencies are installed:
   ```bash
   npm install
   ```
2. Build for development:
   // turbo
   ```bash
   npm run dev
   ```
3. Watch for changes:
   ```bash
   npm run watch
   ```
4. Build for production:
   // turbo
   ```bash
   npm run prod
   ```
